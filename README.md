
# 📊 Excel Sales Dashboard Project

## 📌 Project Overview

This project focuses on creating an **interactive Sales Dashboard in Microsoft Excel** to analyze and visualize business performance. The dashboard helps in understanding key metrics such as revenue, profit, cost, and customer trends across different cities and courses.

The main goal of this project is to transform raw data into meaningful insights using Excel features like charts, slicers, and formulas.


---

## 🎯 Objectives

* Analyze sales performance across different locations
* Track revenue, cost, and profit
* Identify top-performing cities and courses
* Create a dynamic dashboard using slicers
* Present insights in a clear and visual format

---

## 📂 Dataset Description

The dataset includes the following fields:

* Customer ID
* City
* Course
* Revenue
* Cost
* Profit (calculated column)
* Date / Time (if applicable)

---

## ⚙️ Tools & Features Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers (for interactivity)
* Conditional Formatting
* Calculated Columns (Profit, Age, etc.)
* Data Cleaning Techniques

---

## 📈 Dashboard Features

* 📊 Revenue Analysis
* 💰 Profit vs Cost Comparison
* 🌍 City-wise Performance
* 📚 Course-wise Insights
* 🎛️ Dynamic Filtering using Slicers

The dashboard updates automatically based on selected filters (City / Course), making it interactive and user-friendly.

---

## 🔍 Key Insights

* Some cities generate higher revenue but may not have the highest profit due to higher costs
* Certain courses perform better in specific cities
* Profitability depends not only on revenue but also on cost management
* Slicers help in identifying trends quickly for each category

---



## 🚀 How to Use

1. Open the Excel file
2. Go to the Dashboard sheet
3. Use slicers to filter data (City / Course)
4. Analyze charts and insights dynamically
